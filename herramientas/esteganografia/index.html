Esteganografía en Imagen

Herramienta para ocultar o revelar mensajes breves dentro de una imagen. El proceso se realiza localmente en el navegador, sin subir archivos a ningún servidor.

Datos de entrada

Se aceptan PNG, JPG, WEBP y algunos HEIC si el navegador los soporta. La salida siempre será PNG.
0 / 200
La clave se usa para cifrar o revelar el mensaje. Puede tener entre 4 y 16 caracteres.
La imagen no se sube a ningún servidor. Todo el proceso ocurre localmente en este navegador.

Nota

Esta herramienta se proporciona únicamente con fines educativos, de experimentación y apoyo al aprendizaje.

La seguridad depende de la clave utilizada. Una clave corta o fácil de adivinar puede comprometer el contenido. Para conservar el mensaje oculto, mantenga la imagen como PNG y evite convertirla a JPG o enviarla mediante servicios que recompriman imágenes.
